IP Camera Recoder
=================
### NOTE!!! IP Camera for iOS provides this feature, includes audio.
For iOS https://itunes.apple.com/app/id990605467
For Windows http://apps.microsoft.com/windows/app/ip/0b4528ad-cae4-463e-997f-f6265a3364bf

IP Camera Recoder is designed for recording mjpeg stream from APP(likes 'IP Camera for iOS', 'IP Camera for Windows Phone', 'IP Camera for Android', 'IP Camera for Windows 8.1') or other devices, and encode into mpeg4 or avi format.
